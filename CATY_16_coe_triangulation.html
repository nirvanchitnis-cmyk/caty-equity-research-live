2.1 Formula

E(R) = Rf + Œ≤_market √ó (Rm - Rf) + Œ≤_SMB √ó SMB + Œ≤_HML √ó HML

Where:

  • Œ≤_market = Market beta (same as CAPM)
  • Œ≤_SMB = Size factor loading (Small Minus Big)
  • SMB = Size premium (small-cap excess return)
  • Œ≤_HML = Value factor loading (High Minus Low book-to-market)
  • HML = Value premium (value stock excess return)
