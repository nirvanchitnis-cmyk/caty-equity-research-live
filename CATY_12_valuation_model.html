P/TBV Mapping Framework

Core Formula

Gordon Growth Model Simplification

P/TBV = (ROTE - g) / (COE - g)

Where:
- ROTE = Return on Tangible Common Equity (normalized for through-cycle credit)
- COE = Cost of Equity (peer-calibrated from implied COE analysis)
- g = Sustainable growth rate (long-run nominal GDP assumption)

Normalization Methodology

Inputs:
- LTM Net Income: $294.7M
- Average TCE: $2,465.1M
- Average Loans: $19,449.0M
- LTM NCO: 18.10 bps (below long-term average)

Formula:
1. Delta Provision = (42.8 bps - 18.10 bps) √ó $19,449.0M = $48.0M
2. Tax Effect = $48.0M √ó Tax Rate (20.0%) = $9.6M
3. Normalized NI = $294.7M - $48.0M + $9.6M = $256.3M
4. Normalized ROTE = $256.3M / $2,465.1M = 10.40% ‚Üí 10.21%
5. Target P/TBV = (10.21% - 2.5%) / (9.587% - 2.5%) = 1.087x

CATY Residual Analysis

CATY's actual P/TBV of 1.269x is 0.069x below the regression-fitted value of 1.338x, based on its raw ROTE of 10.60%. This suggests CATY is trading at a slight discount relative to its current profitability.
