P/TBV Mapping Framework

Core Formula

Gordon Growth Model Simplification

P/TBV = (ROTE - g) / (COE - g)

Where:
- ROTE = Return on Tangible Common Equity (normalized for through-cycle credit)
- COE = Cost of Equity (peer-calibrated from implied COE analysis)
- g = Sustainable growth rate (long-run nominal GDP assumption)

Normalization Methodology

Inputs:
- LTM Net Income: $294.7M
- Average TCE: $2,465.1M
- Average Loans: $19,449.0M
- LTM NCO: 18.13 bps (below long-term average)

Formula:
1. Delta Provision = (Through-Cycle NCO - LTM NCO) √ó Average Loans
2. Tax Effect = Delta Provision √ó Tax Rate (20%)
3. Normalized NI = LTM NI - Delta Provision + Tax Effect
4. Normalized ROTE = Normalized NI / Average TCE
5. Target P/TBV = (Normalized ROTE - g) / (COE - g)

CATY Residual Analysis

CATY's actual P/TBV of 1.269x is 0.069x below the regression-fitted value of 1.338x, based on its raw ROTE of 11.95%. This suggests CATY is trading at a slight discount relative to its current profitability.

However: This regression uses reported ROTE, not normalized ROTE. When we normalize CATY's ROTE to 10.40% (base case with 42.8 bps NCO), the fitted P/TBV drops to 1.115x, which is below the current market multiple of 1.269x, supporting the HOLD rating.
